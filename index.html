How to create webmaker galleries and add content

This guide uses a fictional gallery called '/bacon' on webmaker.org

  1. Choose a main section of the site (Explore, Make, Teach)
  2. Choose a navigation label for the gallery
  3. Write a main heading. It should be a short phrase that describes the main focus of the gallery.
  4. Write a short, descriptive paragraph to support the heading.
  5. Lead gallery: Login to makeapi.webmaker.org to tag makes for the lead gallery. You should use tags in the following format: 
    webmaker:keyword-#
    where # is the position of the make in the gallery.
  6. The area below the gallery can be filled with makes using tags in the following format: 
    webmaker:keyword
